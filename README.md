跟随b站江协科技学习51单片机过程中写下的代码

[江协科技51单片机入门教程](https://www.bilibili.com/video/BV1Mb411e7re )

基本和视频一致，有几点不同

1. 视频中使用的头文件是**REGX52.H**，我使用的是**STC89C5xRC.H**

2. 我使用的开发板是新版，STC89C52RC，晶振频率和老版不太一样，所以延时和定时器配置有差别
3. 在 *14-1 LCD1602液晶显示屏* 部分，我在实现显示数字的驱动代码时没有选择和江协同样的方式，我直接将其转换成字符串来实现，虽然写起来方便但有些缺陷，比如显示二进制数字我就想不到该怎么用这种方式实现了，再改回江协那种方式又觉得麻烦，最后就干脆不写了
